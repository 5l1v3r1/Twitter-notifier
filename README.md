Twitter-notifier
================


Features
---------
* Commandline base.
* Check last 10 mentions.
* Check last 5 Direct messages.
* Check last followers.
* Show Account's avatar in notification.
* Support KDE, Gnome, etc..


How to use?
--------------
Open Twitter-notifier script, then add your twitter APIs.
